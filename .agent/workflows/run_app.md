---
description: How to run the integrated LMS application
---

1. Start the Backend:
   Open a terminal and run:
   ```bash
   cd c:\Users\shoya\LMS_hack
   python main.py
   ```
   The backend will start at `http://localhost:8000`.

2. Start the Frontend:
   Open another terminal and run:
   ```bash
   cd c:\Users\shoya\LMS_hack\UI
   npm run dev
   ```
   The frontend will start at `http://localhost:5173` (or similar).

3. Access the App:
   Open your browser and navigate to the frontend URL.
   You can log in as an Employee (demo login) and browse the catalog, enroll in courses, and view your learning progress.
